# 🛒 Shopping Cart using Zustand & Context API

A simple React Shopping Cart application built using **Zustand** and **React Context API** for global state management. This project demonstrates how to manage shared state across multiple components without prop drilling.

---

## 📌 Features

- Display a list of products
- Add products to the shopping cart
- Remove products from the cart
- Clear the entire cart
- Global state management using Zustand
- Context API implementation for comparison
- Responsive and clean user interface

---

## 🛠️ Technologies Used

- React.js
- Zustand
- Context API
- JavaScript (ES6)
- CSS3
- Vite

## ⚙️ Installation

### 1. Clone the repository

```bash
git clone https://github.com/your-username/shopping-cart-zustand.git
```

### 2. Navigate to the project folder

```bash
cd shopping-cart-zustand
```

### 3. Install dependencies

```bash
npm install
```

### 4. Install Zustand

```bash
npm install zustand
```

### 5. Run the development server

```bash
npm run dev
```

---

## 🚀 How It Works

- Products are displayed on the home page.
- Clicking **Add to Cart** adds the selected product to the global cart state.
- The Shopping Cart displays all added products.
- Users can remove individual products from the cart.
- The **Clear Cart** button removes all items from the cart.
- Zustand manages the cart state globally, making it accessible from any component without prop drilling.
