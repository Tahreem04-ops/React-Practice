# Day 23 - Advanced SQL (Supabase JOIN + Search)

## 📌 Project Overview

This project demonstrates how to fetch relational data from Supabase using SQL JOIN (relationship queries) and display it in a React application. Users can search orders by customer name or product name.

---

## 🚀 Features

- Display Customers and Orders
- Foreign Key Relationship
- JOIN Query using Supabase
- Search Orders
- Responsive Table UI
- React + Vite
- Supabase Database

---

## 🛠 Technologies

- React
- Vite
- Supabase
- PostgreSQL
- CSS
