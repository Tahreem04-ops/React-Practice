# React Learning Project

## Overview

This repository contains my React learning practice created using **Vite**. The project focuses on understanding the core concepts of React, including component-based architecture, React Hooks, and API integration.

## Technologies Used

* React
* Vite
* JavaScript (ES6+)
* HTML5
* CSS3

## React Concepts Covered

* React Components
* JSX
* Functional Components
* Props
* State Management
* useState Hook
* useEffect Hook
* Event Handling
* Conditional Rendering
* List Rendering
* Fetch API
* Component Reusability

## Project Features

* Modular component structure
* Interactive user interface
* State management using React Hooks
* Data fetching from a public API
* Clean and reusable code
* Responsive layout


## Learning Outcomes

Through this project, I learned how to:

* Create reusable React components
* Build applications using Vite
* Manage application state with useState
* Perform side effects using useEffect
* Fetch and display data from APIs
* Organize React projects efficiently
* Build interactive and dynamic user interfaces
