# Day 26 - Testing with Vitest

## Overview

This project demonstrates unit testing for both a React component and an Express API route using Vitest.

## Technologies Used

- React
- Vite
- Vitest
- React Testing Library
- Express
- Supertest

## Features

- Unit test for a React Button component.
- Unit test for an Express API endpoint.
- Fast and reliable testing using Vitest.

## Installation

### Frontend

```bash
cd client
npm install
npm run dev
```

Run React tests:

```bash
npm test
```

### Backend

```bash
cd server
npm install
npm run dev
```

Run API tests:

```bash
npm test
```
